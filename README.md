# PitchSlap

AI tool that brutally roasts cold emails and rewrites them into better outbound copy.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React
- Gemini 1.5 Flash (`@google/generative-ai`)

## Run

```bash
npm install
cp .env.example .env.local
# set GEMINI_API_KEY in .env.local
npm run dev
```

Open `http://localhost:3000`.
